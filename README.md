# Integrated Lung Disease Detection System

A unified AI-based diagnostic platform for detecting multiple lung diseases
using Chest X-rays and patient symptoms.

## Diseases Covered
- Asthma (symptom-based ML model)
- Lung Cancer (CNN – PyTorch)
- Pneumonia (Deep Learning – TensorFlow)
- Tuberculosis (Deep Learning – TensorFlow)

## Tech Stack
- Python
- Streamlit
- PyTorch
- TensorFlow / Keras
- XGBoost
- Scikit-learn
- Pandas, NumPy

## Features
- Upload chest X-ray for analysis
- Symptom-based asthma severity prediction
- Confidence-based disease assessment
- Unified diagnostic dashboard
